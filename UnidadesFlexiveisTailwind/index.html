Guia de Unidades Flexíveis em CSS

Unidades flexíveis (ou relativas) em CSS são essenciais para criar layouts responsivos. Elas se adaptam a diferentes tamanhos de tela e configurações do usuário, ao contrário das unidades fixas como pixels (px).

1. rem (Root EM)

A unidade rem é relativa ao tamanho da fonte do elemento raiz (<html>). Nesta página, a fonte raiz foi definida como 62.5% do padrão, o que resulta em 1rem = 10px, facilitando os cálculos.

Este parágrafo tem font-size: 1.8rem, o que equivale a 18px.

2. em

A unidade em é relativa ao tamanho da fonte do seu elemento pai. Isso pode levar a um efeito cascata se houver muitos elementos aninhados.

O font-size deste contêiner é 2rem (20px).

Este parágrafo interno tem font-size: 1.5em, resultando em 1.5 * 20px = 30px.

3. % (Porcentagem)

A unidade % é sempre relativa a uma propriedade do seu elemento pai. É fundamental para criar layouts fluidos.

Este elemento tem width: 75% do seu contêiner cinza.

4. vh (Viewport Height)

1vh é igual a 1% da altura da viewport (a área visível da janela do navegador). É perfeito para criar seções que ocupam a tela toda.

Este bloco tem height: 40vh.

5. vmin e vmax

vmin usa a menor dimensão da viewport (altura ou largura) e vmax usa a maior. Redimensione sua janela para ver o efeito.

30vmin
20vmax

6. ch (Character Unit)

A unidade ch é relativa à largura do caractere "0" da fonte. É excelente para controlar o comprimento de linhas de texto para melhor legibilidade.

Este bloco de texto tem a propriedade width: 65ch;. Isso limita sua largura para conter, em média, 65 caracteres por linha, tornando a leitura mais confortável.
